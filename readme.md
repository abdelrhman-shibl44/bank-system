# Bank System (Learning Project)

## Project Overview

This is a small console-based bank system implemented in C++ as a hands-on learning project. It was created to strengthen foundational programming skills across languages, with a focus on C++ for systems-level concepts.

## Motivation

I'm learning programming to build a stronger foundation across languages and paradigms. Working in C++ helps with systems programming, memory management, while the project also improves general skills like problem-solving, algorithms, data structures, software design, and debugging.

## Files

- `BankProject.cpp` — main program source file
- `mylib.h` — helper/header functions used by the project
- `Clients.txt` — sample data file for client records

## Build (Windows, using g++)

Open a terminal in the project directory and run:

```
g++ -std=c++17 BankProject.cpp -o BankProject.exe
```

## Run

After building, run the executable:

```
.\BankProject.exe
```

## Skills & Notes

- Skills gained: C++, algorithms, file I/O, debugging, and basic software design.
- This repository is a learning project and can be extended with features like transactions, improved persistence, and stronger input validation.
